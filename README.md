# CRUD Users Full Stack

API REST Full Stack para gerenciamento de usuários utilizando Node.js, Express e PostgreSQL.

## 🚀 Tecnologias

- Node.js
- Express
- PostgreSQL
- JavaScript
- Dotenv
- Nodemon

## 📌 Funcionalidades

- Criar usuário
- Listar usuários
- Buscar usuário por ID
- Atualizar usuário
- Deletar usuário

## 📂 Estrutura do Projeto

backend/
├── src/
│ ├── config/
│ ├── controllers/
│ ├── routes/
│ └── server.js

## 🔗 Endpoints

GET /users

GET /users/:id

POST /users

PUT /users/:id

DELETE /users/:id

## ⚙️ Como rodar o projeto

### Instalar dependências

npm install

### Rodar servidor

npm run dev

## 🗄️ Banco de Dados
